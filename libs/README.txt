This project has two external library dependencies:

1. `ActionBarSherlock` and `ActionBarSherlock MapView Plugin`, to add actionbar support to API < 11 
	(pre-honeycomb).
	
		http://actionbarsherlock.com/

2. `Android MapView Balloons` to display labels on map

       https://github.com/jgilfelt/android-mapviewballoons
